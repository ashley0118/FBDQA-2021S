**一个交易系统的要素**

| 要素     | 解释                       |
| -------- | -------------------------- |
| 市场     | 买卖什么                   |
| 入市     | 什么时候买入               |
| 退出     | 什么时候退出一个盈利的头寸 |
| 止损     | 什么时候放弃一个亏损的头寸 |
| 头寸规模 | 买卖多少                   |
| 战术     | 怎么买卖                   |
